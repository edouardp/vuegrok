## Installing 

### Ubuntu

```
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## Installing vue-cli

### Ubuntu

```
npm install vue-cli
```

then set your path to include the ~/node-modules .bin directory

```
export PATH=$(npm bin):$PATH
```
